Kapitel 1

Punkte im Raum

Ein Punkt P im Raum hat drei Koordinaten: P(x₁ | x₂ | x₃). Drehe die Szene und beobachte, wie die Koordinaten den Punkt im Raum festlegen.

3D-Koordinatensystem mit Punkten
🖱 Ziehen = drehen · Scroll = zoomen
Abstand zweier Punkte
d = √[ (q₁−p₁)² + (q₂−p₂)² + (q₃−p₃)² ]
3D-Pythagoras immer positiv P(p₁|p₂|p₃) → Q(q₁|q₂|q₃)
Mittelpunkt einer Strecke
M = ( (p₁+q₁)/2 | (p₂+q₂)/2 | (p₃+q₃)/2 )
Koordinaten mitteln liegt genau in der Mitte
📍

Koordinatenebenen

Liegt ein Punkt auf der x₁x₂-Ebene, ist seine x₃-Koordinate = 0. Auf einer Achse sind die anderen beiden Koordinaten 0.

z.B. P(2|3|0) liegt in der x₁x₂-Ebene
🧮

Achsensystem

x₁ zeigt nach vorne, x₂ nach rechts, x₃ nach oben. Die drei Achsen stehen senkrecht aufeinander — orthogonales System.

Rechte-Hand-Regel
🧪 Quiz: Welche Koordinate ist 0, wenn P auf der x₂-Achse liegt?
✓ Richtig! Liegt ein Punkt auf einer Achse, sind genau die beiden anderen Koordinaten null.
✗ Nicht ganz. Auf der x₂-Achse gilt: x₁ = 0 und x₃ = 0, nur x₂ bleibt frei.

Verbindungsvektor — „Spitze minus Fuß"
AB⃗ = B − A = (b₁−a₁, b₂−a₂, b₃−a₃)
Betrag eines Vektors
|v⃗| = √(v₁² + v₂² + v₃²)
Länge des Pfeils = Abstand Start → Ende

Ortsvektor

Geht vom Ursprung O(0|0|0) zum Punkt P. Jeder Punkt bekommt so seinen „eigenen" Vektor.
Schreibweise: p⃗ = OP⃗

Richtungsvektor

Gibt nur eine Richtung vor — kein fester Startpunkt. Kann beliebig parallel verschoben werden. Wichtig bei Geraden.

Gegenvektor

Der Gegenvektor zu a⃗ ist −a⃗. Gleich lang, gleiche Achse — aber entgegengesetzte Richtung.

Richtungsumkehr

Kollinearität

Zwei Vektoren sind kollinear (parallel), wenn einer ein Vielfaches des anderen ist:
a⃗ = r · b⃗

a⃗ = r · b⃗
🧪 Quiz: Was ergibt der Betrag von v⃗ = (3, 0, 4)?
✓ Richtig! √(9 + 0 + 16) = √25 = 5. Ein klassischer 3-4-5-Pythagoras!
✗ Denk nochmal: |v⃗| = √(3² + 0² + 4²) = √(9+0+16) = √25 = 5.
Kapitel 3

Rechnen mit Vektoren

Addition, Subtraktion, Skalarmultiplikation — geometrisch und rechnerisch verstehen.

Vektoroperationen in 3D
🖱 Drehen um zu sehen wie die Pfeile zusammenpassen
➕

Addition

Pfeile aneinanderhängen. Koordinatenweise addieren.

a⃗ + b⃗ = (a₁+b₁, a₂+b₂, a₃+b₃)
Geometrisch: Pfeile hintereinander
➖

Subtraktion

Gegenvektor anhängen: a⃗ − b⃗ = a⃗ + (−b⃗)

a⃗ − b⃗ = (a₁−b₁, a₂−b₂, a₃−b₃)
Geometrisch: Differenzpfeil
✖️

Skalarmultiplikation

Jede Koordinate mit dem Skalar r multiplizieren. Streckt, staucht oder dreht den Pfeil.

r · a⃗ = (r·a₁, r·a₂, r·a₃)
r>1: strecken · 0<r<1: stauchen · r<0: umkehren
∥

Kollinearität prüfen

Gibt es ein r, sodass a⃗ = r · b⃗? Dann sind die Vektoren parallel (kollinear).

r = a₁/b₁ → Prüfe ob = a₂/b₂ = a₃/b₃
Quotienten vergleichen
🧪 Quiz: Was ist 3 · (1, −2, 0)?
✓ Richtig! Jede Koordinate × 3: (3·1, 3·(−2), 3·0) = (3, −6, 0).
✗ Jede Koordinate wird mit 3 multipliziert: 3·1=3, 3·(−2)=−6, 3·0=0 → (3, −6, 0).

🧪 Quiz: Gegeben ABCD ist ein Parallelogramm mit A(1|0|0), B(3|1|0), C(4|3|0). Wo liegt D?
